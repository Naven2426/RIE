## Overview

**Reality Integrity Engine (RIE)** is a modular AI-powered platform designed to detect, analyze, and verify the authenticity of digital media, including images, videos, and audio.

As generative AI technologies continue to improve, identifying manipulated content has become increasingly difficult. RIE addresses this challenge by combining modern web technologies, computer vision, machine learning, and forensic analysis into a scalable verification platform.

The project follows a microservice-oriented architecture where the frontend, backend, and AI inference engine operate independently while communicating through secure REST APIs.

---

## Vision

To build a trusted, scalable, and enterprise-ready platform that helps individuals and organizations verify digital media authenticity and reduce the spread of AI-generated misinformation.

---

## Problem Statement

Synthetic media generated by modern AI models has become increasingly realistic. Deepfakes, voice cloning, manipulated images, and fabricated videos present significant risks to journalism, law enforcement, cybersecurity, education, and public trust.

Existing solutions are often specialized for a single media type or require significant technical expertise. Reality Integrity Engine aims to provide a unified platform capable of analyzing multiple forms of digital media while presenting results through an intuitive user interface.

# System Workflow

1. User uploads media through the web application.
2. Backend validates and stores the uploaded file.
3. Backend forwards the media to the AI Engine.
4. AI Engine performs forensic analysis.
5. Detection results are returned to the backend.
6. Backend stores the analysis.
7. Frontend displays an integrity report with confidence metrics.

# Repository Setup

The project consists of three independent services:

* Frontend (React)
* Backend (Express.js)
* AI Engine (FastAPI)

Each service should be configured separately.

---

# Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

The frontend will be available at:

```
http://localhost:5173
```

---

# Backend Setup

```bash
cd backend

npm install

npm run dev
```

Default backend address:

```
http://localhost:5000
```

---

# AI Engine Setup

Create a virtual environment:

```bash
cd ai-engine

python -m venv venv
```

Activate the environment.

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the inference server:

```bash
uvicorn app:app --reload --host 127.0.0.1 --port 8000
```
